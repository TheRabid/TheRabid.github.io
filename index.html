Proyectos recientes

Edición cinematográfica y continuidad narrativa en realidad virtual

Este proyecto de investigación tiene como objetivo el estudio de la atención de los usuarios en cinematografía en realidad virtual. ¿Se mantiene la percepción de continuidad en entornos inmersivos? ¿Qué influencia tienen determinados factores en la atención de un usuario en realidad virtual?
Entre otras cosas en este proyecto se llevaron a cabo una serie de experimentos empíricos sobre la atención de los usuarios en narrativa en realidad virtual, con el fin de comprobar la efectividad de las técnicas de cinematografía tradicional en un entorno inmersivo.
Este proyecto de investigación fue aceptado en la conferencia ACM SIGGRAPH 2017, que publica en la revista ACM Transactions on Graphics (Q1, 1/106)

FelinoTweets

FelinoTweets es una plataforma web para gestionar múltiples cuentas de Twitter de una forma fácil y simple. Puedes asociar tus cuentas de Twitter y realizar diversas operaciones como enviar tweets, responder tweets, programar tweets, visualizar estadísticas en tiempo real sobre tus tweets, etc. Utiliza NodeJS, AngularJS y una base de datos MongoDB con Mongoose.
